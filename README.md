## vue2.x+ vux+ vue-router+ mockjs+ webpack打包多页面应用

#### 本例根据bluefox1688开源的进行的二次修改，这里只做了几个有代表性的页面，实际页面可根据需求进行开发

> 运行方式

	npm install
	npm run dev
	npm run build
